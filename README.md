## Empezando

Para usar el proyecto en tu computadora.



### Prerrequisitos


```
NodeJS
```

### Instalación

Descargar el proyecto o clonarlo

Usar el instalador de npm para obtener modulos

```
npm install
```

Correr servidor de angular

```
ng serve -o
```
